Bienvenidos



Este proyecto fue creado para ser presentado ante el equipo de FXInformática como parte de su proceso de ingreso al área de desarrollo, la misma fue implementada con:

  • Net.5/Entity Framework.
  • SQL server para la base de datos.
  • Angular y TypeScript para la parte del frontend.
  • CSS, Bootstrap y Angular-Material para los estilos.


La utilización e implementación de la misma es bastante sencilla. Básicamente es una herramienta para inscribir jugadoras y crear equipos.
El sistema permite también acceder a cada jugadora, muestra su ficha, así como también permite editarla o borrarla de la aplicación.
Lo mismo sucede con los equipos, se puede acceder a ellos para verlos e incluso editarlos. En cuanto a borrar un equipo, primero debemos deshacernos de sus integrantes.



A continuación, puede crear jugadores desde el botón Jugadoras, o crear equipos desde el botón Equipos.
